# 🎓 Student Performance AI

## 📌 Overview
This project is a beginner machine learning model that predicts student performance based on study habits.

## 🧠 Problem Statement
Can we predict a student's score based on how many hours they study?

## ⚙️ Tools & Technologies
- Python
- NumPy
- Scikit-learn
- Google Colab

## 📊 How It Works
The model uses Linear Regression to learn from sample data (study hours vs scores) and predict future outcomes.

## 🚀 Sample Prediction
If a student studies for 6 hours, the model predicts a score of approximately 105.

## ⚠️ Note
This is a basic model and does not limit predictions within realistic exam scores (0–100). Future improvements will address this.

## 🎯 Learning Outcome
- Understanding of basic machine learning concepts
- Model training and prediction
- Hands-on experience with AI tools
